# Hangman

Консольная игра "Висилица"

## Правила игры
В начале игры компьютер загадывает слово, а вы пытаетесь угадать это слово, по-очереди вводя буквы(не прописные). У вас будет 6 попыток. После каждой неудачной буквы компьютер будет дорисовывать висилицу. Отгаданные буквы и количество ошибок также будут отображаться на экране. После окончания игры вы вернетесь в меню.
